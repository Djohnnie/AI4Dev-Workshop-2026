description: Example skill file for reviewing the Snake lab against the workshop checklist.

1. Read `instructions/copilot-instructions.md` to understand the expected gameplay.
2. Inspect `Program.cs` and `wwwroot/` to confirm the app is a tiny ASP.NET Core static-files host.
3. Verify the core behaviors explicitly:
   - arrow-key movement
   - spacebar start, pause, and restart
   - wrap-around movement
   - growth on food
   - self-collision game over
   - score and status updates
4. Prefer feedback tied to the workshop checklist over generic code review comments.
5. If the implementation is close, suggest the smallest next fix first.
