# User Stories

## Epic 1 — Link creation

### Story 1.1

As a workshop facilitator, I want to submit a long destination URL so that I can share a shorter, easier link in slides or chat.

### Story 1.2

As a workshop facilitator, I want to request a custom alias so that memorable short links can be used in demos.

## Epic 2 — Link redirection

### Story 2.1

As a visitor, I want a short URL to redirect me to the underlying destination so that I reach the intended page without copying a long link.

### Story 2.2

As a visitor, I want a clear response when a short code does not exist so that I know the link is invalid rather than broken silently.

## Epic 3 — Link statistics

### Story 3.1

As a facilitator, I want to see how many times a short link has been used so that I can confirm engagement during a workshop.

### Story 3.2

As a facilitator, I want to see when a link was last used so that I can tell whether a redirect happened recently.

## Epic 4 — Workshop operability

### Story 4.1

As a participant, I want the project to run locally without extra services so that I can focus on analysis and implementation instead of environment setup.

### Story 4.2

As a participant, I want the codebase to stay small and layered so that I can map analysis decisions to the final implementation.
